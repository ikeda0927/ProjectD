PushとPull
ローカルのリポジトリとリモートのリポジトリとを繋げた後の手順を記します。
まず、
localリポジトリにコマンドプロンプト等で移動して、
git pull origin master
を実行します。
そうするとリモートリポジトリ内のファイルがローカルにコピー(同期)されます。
この状態でファイルの変更や新規ファイルの追加などを行います。
変更や追加をしたら、そのファイルをリモートリポジトリに更新します。
例として、test.txtを変更したとします。
git add test.txt
を実行することでtest.txtをステージにセットします。
git commit -m "コメント（変更者の名前とか変更箇所とか）"
でコミットします。
git push origin master
でプッシュします。この時にGitHubのユーザー名とパスワードを聞かれるので入力します。
これでエラーが出なければ更新完了です。
